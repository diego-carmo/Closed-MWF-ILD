# Closed-Form Solution to the Multichannel Wiener Filter with Interaural Level Difference Preservation
Audio repository for the results of the paper "Closed-Form Solution to the Multichannel Wiener Filter with Interaural Level Difference Preservation". Above there is the audios in mp4 format related with an acoustic scenario for a speech source in 0° and the interfering in -60°. The full set of audios in wav format related with an interfering source in -90°, -60°, -40°, 40°, 60°, 90° can be download from [Audios.zip](https://github.com/diego-carmo/Closed-MWF-ILD/files/9364850/Audios.zip)
